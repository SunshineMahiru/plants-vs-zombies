#### Java版植物大战僵尸
1. 开发工具：Eclipse IDE
2. 所用技术：java基础语法、多线程、I/O、面向对象程序设计思想、Java图形编程、
3. 主要功能：实现了游戏的成就、商店、图鉴、花园界面。开始游戏后，不定时会有不同种类僵尸从右方出现向左移动、阳光随机落下，收集阳光可在草地上种植不同植物，部分植物可发出子弹攻击僵尸，僵尸靠近植物可缓慢吃掉植物，僵尸突破植物防线进入最左侧则游戏结束。
